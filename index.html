Cemitec

Um grupo de alunos da Etec Rosa Perrone Scavone.

  • Ir para Diário de Bordo

O projeto é construido como um software web de mapeamento de cemitério e Banco de dados, um site voltado para a secretaria do cemitério e facilitação do fluxo de visitantes e atividade dos funcionários. Haverá um Banco de Dados contendo todos os registros de óbito armazenados pela secretaria, e registros de mapa do local.

  • Learn More

Com dados completos de óbito, o funcionário da secretaria irá inserir o nome completo do falecido, e o sistema aplica a "filtragem" desses dados, para assim obter o registro de tal ente específico. Obtendo essa informação, o site pode aplicar um recurso de um mapa local para orientar o visistante para qual lápide/tumulo está o ente-queriso que ele procura, sinalizando com um icone.
